成本

经济学中的成本和会计学中的成本不一样，分别称为经济成本和会计成本，详细就不说了，很简单。

分类

  • 显性成本和隐性成本

  • 固定成本和可变成本

  • 沉没成本
    覆水难收了解一下。

  • 私人成本和社会成本

社会成本=私人成本+外部成本

由于生产的社会性，单个消费者或生产者的行为不可避免地会对其他人的福利造成影响，也就是说会产生外部性（外部效应）

1.某一生产者的经济活动对他人产生了有利的经济影响，但是自己却未能从中获取报酬，此时便产生了生产的正外部性。比如果农为养蜂人提供了制造蜂蜜的原料，但是没有收费。

2.但是如果生产者的经济活动使他人付出了代价但是又没有为此进行补偿就产生了生产的正外部性。比如造纸厂污染了水资源，但是却没有为此进行补偿。

3.教育好自己的孩子对隔壁邻居以及社会都产生了正外部性；修建自家草坪邻居也得到了不用支付报酬的好处。这是从消费者的角度来说的

4.吸烟者的行为危害了被动吸烟者的身体健康，这属于消费者的负外部性。
  • 短期成本与长期成本

发现固定成本和可变成本，短期与长期成本在生产里面也出现过，其实，它们的概念区分也是类似的，可以说是几乎一样，只是把生产要素这个名字变为了成本，这两者本身就是同一种东西，只是放在两个不同的角度来说。

生产理论

Posted on 2018-06-07 | In Course

导论

从生产角度分析厂商的供给，讨论在不同生产周期里厂商在要素价格既定和投入既定的情况下，如何进行要素投入和规模选择，使自己生产行为最优化，实现既定投入下产出最大或既定产出下投入最小。

主要概念：

短期、长期分类、边际生产力递减、规模经济、范围经济、规模报酬

在这里把厂商的生产目标定的是支出最小化或者是收入最大化，因此在一些计算体中问的也是厂商的成本和产量。

短期生产

  • 短期生产周期中部分要素可变（可变成本），部分要素不可变（固定成本）。不可变的部分正是因为时间太短，短到来不及改变。在长期生产中，所有要素都有时间来做出改变。

  • 依然是为了讨论简便，这里将短期生产函数中以资本（K）和劳动（L）代指固定生产要素和可变生产要素的投入。
    $Q=f(·K,L)$ （点表示K不变，其实在一般的函数中表示出来的只有L）

总产量：$Q=f(L)$

平均产量：$AP L=Q/L$

边际产量：$MP L=dQ/dL$

这三个函数对应的曲线之间的关系：

边际产量最高点就是总产量斜率的最大值；
边际产量交于平均产量线的最高点，在这点之前，边际产量虽然先增后减但是仍然是 大于平均产量的，因此在这点之前，平均产量递增，这点之后平均产量递减。

  • 这张图就展现了三条曲线的关系，但是从这幅图中还能得出的一个结论就是：哪个生产阶段对于厂商是最好的，这就引出了三个生产阶段————

    • 边际产量最高点到平均产量最高点为第一阶段

    • 平均产量出现下滑趋势，但是在总产量下滑之前为第二阶段

    • 总产量下滑之后为第三阶段

短期生产中最优生产阶段

很明显，在一开始就说了厂商追求的产出最大或投入最小，即产量最大或生产要素数量最小（因为两者的价格都是既定的），从图中可以看出来，在第二阶段持续下去就可以达到产量最大。所以最适合的生产阶段就是第Ⅱ阶段。

在第一阶段厂商增加劳动投入就可以使平均产量增加，那么厂商为了追求更高的产量一定能会追加劳动投入，到了追加到一定程度边际产量会减少到平均产量之下，此时平均产量开始下跌，但是由于不变要素的利用率可以弥补可变要素生产效率的降低，总产量仍呈递增趋势，所以厂商在第二阶段仍然会增加劳动投入继续生产。显然，第三阶段投入增加但是产出减少。

  • 在长期生产中所有要素都是可变的，并且从生产技术的角度两种生产要素都是可以互相替代的。（长期生产函数和短期生产函数一样也是关于生产要素L、K的函数，只不过这里的K和L一样是可变的）

消费者行为理论

Posted on 2018-06-02 | In Course

效用

  • 效用是一种主观的东西，是消费者在消费行为中的主观感受。

  • 消费者每多买一件相同的商品，他所收获的满足感就会下降一点，这种得到的满足感就是消费者多买这一件商品得到的边际效用（$MU$）；这种现象称为边际效用递减规律，这种递减规律也还体现在货币中，叫做货币的边际效用递减规律。

  • 除了边际效用，还有总效用。看下面这张图就应该知道总效用的意义以及它是怎么算了。

  • 效用也分为基数效应和序数效用。

顾名思义，基数效用就是可以用具体数字表示出来效用的多少，序数效用就是只能用序数（第一、第二…）来表示效用的大小程度。

按照逻辑来讲，叙述效用更适合表达消费者的心理，因为消费者的效用属于一种主观感受是无法用具体数字来表示的。

但是，在运算当中，还是基数效用更加简便。所以在研究消费者均衡时，序数效用下的消费者均衡就不是用计算的方法，而是作图的方法。

预算线

  • 预算线这一概念就和消费者的主观一点关系都没有了，它的（斜率、位置）变动只和价格、收入有关。预算线上的每一点都是在既定收入水平和价格水平下购买的两种商品的各种可能最大数量组合（在这里假设X、Y代表所有的商品）

  • 知道商品的价格和数量就可以算出预算线的方程了，比如6X+8Y=240（6、8为价格，X、Y为两种商品的数量）。

  • 预算线的变动很简单就不说了这里。

无差异曲线

  • 无差异曲线是用来表示给消费者带来同等程度的满足水平或效用水平的两种商品的各种不同组合的轨迹。

  • 它的主观性也非常强，它的位置、形状都是由消费者的偏好决定的

  • 无差异曲线也是在序数效用下研究消费者均衡的工具。

  • 特点

同一坐标平面上有无数条无差异曲线。每一条无差异曲线代表的是一种效用水平，位置越低表示效用水平越低。

任何两条无差异曲线的不会相交，因为它们表示的是不同的效用水平。

曲线向右下方倾斜。——（经济意义）消费者消费两种商品的数量反向变化。

曲线凸向原点。——曲线斜率绝对值是递减的。因为边际效用递减规律的作用下，消费者愿意用一种商品代替另一种商品的意愿会逐渐下降。也就是边际替代率（MRS）逐渐下降——用一种商品代替另一种商品的比率逐渐下降。

$MRSXY=-▲Y/▲X=MUX/MUY$

$-▲Y/▲X(▲X→0)=dY/dX$

消费者均衡

上面也说了，讨论消费者均衡要讨论基数效用下的均衡和序数效用下的均衡两种。

  • 首先消费者均衡就是消费者付出的金钱的效用和商品带来的的效用等价的情况。假设每花一个货币得到的效用是r，商品的价格是P，购买这件商品的边际效用为MU。那么所付出的货币的效用为：$Pr$，要想得到消费者均衡的结果就要满足一个等式：$Pr=MU$（r=MU/P;P=MU/r）。

基数效用下的消费者均衡

  • 首先有两个条件需要牢记：

1.消费者花费一单位货币无论购买哪种商品的边际效用都相等时是消费者均衡的状态。（因为任何改变都只能使消费者所得的总效用减少。）

在这个条件的基础下，假设消费者购买商品X、Y、Z，则应分别满足：

$r=MUX/PX = MUY/PY = MUZ/PZ;$

2.消费者购买各种商品的边际效用之比等于价格之比。

$MUX/MUY=PX/PY$

如果$MUX/PX > MUY/PY$,那么意味着单位货币花在商品X上能给消费者带来更大的边际效用，因此，理性的消费者会买性价比更高的商品X代替Y，但是这替代不会一直进行下去，因为在替代的过程中商品X的边际效用会下降，而商品Y的边际效用随着数量减少反过来递增，此消彼长使得$MUX/PX$逐渐趋于$MUY/PY$,反之亦然。

序数效用下的消费者均衡

  • 条件：

1.在既定的收入水平和商品价格下，消费者能获得最大的效用。

  • 序数效用是用无差异曲线来表示的，消费者均衡点是像下图中的无差异曲线和预算线的交点（当预算线作为无差异曲线的切线时），即在既定的收入水平和商品价格下，消费者所能获得的最大效用的商品组合。

  • 切点处$MRSXY=MUX/MUY=PX/PY$ （这里发现与采用基数效用分析消费者均衡的条件是一致的）

需求供给理论——均衡价格理论前奏

Posted on 2018-05-26 | In Course

需求理论

定义

在定义中需要明确需求是一种关系，是在以特定时期内，一定价格与在此一定价格之下消费者愿意且能够购买的商品数量之间的关系；而需求量是一种具体的值，是消费者愿意且能够购买的商品的数量。

需求规律

  • 价格和需求量成反比。
  • 但是奢侈品和吉芬商品不符合此规律。

    替代品：一种商品的价格与另一种商品的需求正向变动；
    互补品：一种商品的价格和另一种商品的需求反向变动。

需求函数

很多因素会影响需求数量，比如P(价格)Pr（相关商品的价格）、M（收入水平）、F（对未来收入的预期）……为分析简单，就只分析价格（P）一种因素
$Q=f(P,Pr,M,F……)$
$Q=f(P)$

如果价格不变，只考虑相关商品对需求量的影响，有需求交叉函数：
$Q=f(Pr)$

类似还有需求收入函数

需求与需求量的变动

  • 需求的变动是需求曲线的移动，而需求量的变动是需求曲线上的点在曲线上的移动。

  • 需求变动是受到价格之外的因素影响，需求量的变动是只受到价格影响引起的需求量的变动。

需求弹性

有点弹性和弧弹性两种，点弹性用到微分的形式，但其实两个式子的形式是差不多的。

  • 需求价格弹性

是需求对于价格变化的反应程度，弹性系数等于需求量变动的百分比除以价格变动的百分比。
对于线性需求而言，斜率不变，但是弹性是一直变动的，因为P/Q在不同的位置值不一样。

  • 分类

富有弹性$ed$>1 需求量对价格的反应比较敏感,降价可以增加销售收入

缺乏弹性$ed$<1 降价会减少收入

单位弹性$ed$=1， 这时价格变动1%引起的需求量变动也是1%，价格变化对于销售收入没有影响

没有弹性$ed$=0，其需求线与纵轴平行

完全弹性$ed$趋于无穷，其需求线与横轴平行

吉芬商品和炫耀性商品的需求价格弹性大于0

  • 需求交叉价格弹性

大于0替代品

<0互补品
=0无关商品

  • 需求收入弹性

与需求价格弹性定义、算法类似

$eM$>1,奢侈品,需求对于收入变化较为敏感
$eM$<1,必需品
$eM$<0,抵档商品，当收入增加时会减少对此类商品的需求，但是到后面会知道在替代效应和收入效应共同影响之下，抵档商品的总效应为正。

微观经济学(一)

Posted on 2018-05-24 | In Course
  • 为什么边际成本线高于平均总成本曲线时，平均总成本曲线呈上升趋势？

MC是一条神奇的曲线，它交与ATC和AVC的最低点。边际成本大于平均总成本时，平均总成本就会上升，或者换一种说法：当边际成本经过平均总成本曲线后，边际成本的上升带动了平均总成本的上升。（随着边际产量先递增后递减的规律，边际成本和平均可变成本也先递减后递增）想一下，当平均总成本为一个值时，边际成本是多生产一个产品所产生的成本，当边际成本增加时，平均总成本也跟着增加。

  • 在完全竞争市场下，价格由市场决定企业无法决定，因此价格表现在图上就是一条不变的水平直线，但是企业终归是要寻求利益最大化即利润最大化。在$P=MR=MC$时利润实现最大化，推导过程如下。

企业的利润计算公式 $L=P * Q-C(Q)$

要求最大利润，对R求导 $L’=P-dC/dQ$

令$R’=0$ 有 $P=dC/dQ=MC$

这表明P=MC时利润最大，此时的产量可使利润最大化，而在完全竞争市场下价格不变，因此边际收益MR=P。

所以 $P=MR=MC$

    面向对象是基于面向过程而言的，可以把复杂的事情简单化，让我们从执行者变为指挥者每提高了代码的复用性。
    例如：烧水（面向过程：找到烧水壶——打开盖子——装水——关闭盖子——打开电源——水烧好了） （面向对象：拿壶——装水——烧水——等待——水烧好了）

  • 类与对象之间的关系？
    类就是有很多相同特性的分类，而对象就是一类中具体的一个。

  • 如何对类进行分析，如果创建自定义类对象，并如何指挥对象做事情？
    首先分析这个类是个什么样的事物；其次分析这个类具有哪些属性和行为；把这个事物的属性和行为对应成我们的成员变量和成员方法。
    通过new关键字创建这个类的对象。 类名.属性名; 类名.方法名（）；直接调用。

  • 成员变量和局部变量的区别？
    作用范围
    成员变量：作用于整个类中。 局部变量：作用于所在区域。
    生命周期
    成员变量：随着类的创建而存在，随着类的消失而消失，执行完语句后等JVM空闲时由垃圾回收机制清理。
    局部变量：是创建对象时才存在的，执行完回立即释放；所以成员变量的生命周期更长。
    初始值
    成员变量：存在于堆内存中，有默认初始值。 局部变量：存在于栈内存中，没有默认初始值，需要手动赋值。

Java中函数的定义

Posted on 2018-05-07 | In Java
  • 函数的格式

修饰符 返回值 类型函数名(形式参数类型1 参数名1,形式参数类型2 参数名2,形式参数类型3 参数名3……)

{

函数体语句;

return 返回值;

}

  • 修饰符：public 公共的static 静态的

  • 返回值类型：就是函数所实现的功能结果的数据类型。

  • 方法名：自己定义，符合标识符命名规则就行；驼峰命名规则 :单个单词首字母小写，多个单词后面的单词首字母大写。

  • 参数列表：即形式参数列表，用于接收实际参数。

  • 参数类型：参数的数据类型

  • 参数名：即变量名

  • 函数体语句：就是函数要完成的功能

  • return:用来结束函数

  • 返回值：即这个函数的功能的结果 ，由return带回调用地方。

    注意:void 表示不确定返回值类型，不能理解为没有返回值，只是返回值的类型不确定而已;

    此外，返回值类型为void时，由于其返回值类型不能确定，故不能定义形式参数变量来接收；

    当返回值类型为void时，return可以不写；但是当返回值类型为其他如int String等时就必须写，否则编译会出错。

  • 自定义一个函数的三步

    • 确定返回值类型

    • 确定方法名

    • 明确参数列表

  • 函数的参数传递

    • 形式参数：函数定义时的参数为形式参数——>简称形参

    • 实际参数：函数调用时的参数为实际参数 ——>简称实参

      参数传递：只能有实参传给形参

      注意：实参和形参在个数、顺序、类型上都要保持一致
