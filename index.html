滚柱上的石块

发表于 2017-11-29 | 分类于 Secret

有周长一米的滚柱。这个滚柱旋转一周时，放在上面的石头前进多少米。（滚柱与地面接触）

这道题看似是一道物理题，但是是我在关于数学论辩谜题的书上看到的。不过我物理一向不好，并且高中学的是文科，只学了半年的物理，所以对这种题不太想用物理的方法解决。

经过我的思考，得出解决这个问题的答案是：当滚柱不与地面接触时转动滚柱一周，滚柱上面的石块会移动一米（这个可以亲自实验一下：拿一根笔，一张手帕纸），此时滚柱位置没有变化，石块在滚柱转动的过程中受到的力使它的位置前进了一米。当滚柱在地面上滚动时，使滚柱自身旋转的力并没有发生变化，所以这个力会传递给石块使其移动一米，但是因为在地面，滚柱的相对位置就发生了变化，上面的石块也会跟着移动一米。所以石块随着滚柱整体移动一米，同时在滚柱转动的过程中石块有比滚柱多移动了一米，加起来就是两米。

这个好像用物理学过的方法解决更简单一点。不过在作者给的答案中说和在电动扶梯上行走的道理类似，这也是一个很不错的类比。

看霍金微博有感

发表于 2017-11-24 | 分类于 Person

在世界变得更糟之前，尽自己的力量，尽自己的生命去为整个世界做贡献。感谢英语课上让我了解到了霍金，这个完美的人。希望更多的年轻人投入到科学当中，如果没有这方面的能力，那就在自己擅长领域做出贡献，发现解决问题，而不是只能叹气。让世界变得更美好，不能直接研究科学，那就间接地来。生活在果壳中，但又确（相信自己）是宇宙之王。这是有多么宏伟的格局才敢想的，宇宙做你的背景当之无愧。

在此，致敬所有的为人类世界做出贡献的伟大科学家们。

庄周梦蝶在中国从古至今已经流传了几千年大概。但是没有人将它与科学联系在一起。科学中极其重要的是想象力，真是难以想象霍金和庄周在一块会发生怎样的奇迹的发现。虽然他们研究的方向不一样，但是庄周应该会带给霍金以不凡的灵感。而庄周那颗向往自由的心或许会飞到外太空吧。

最后深深地感到自己的理解与学习能力的缺乏。

不是独裁者的领导者

发表于 2017-11-23 | 分类于 Audio

每天听本书《探寻式领导》

每回听完一本书结尾都会有一个总结，这本书的总结如下：

  • 一要问；

  • 二要问好问题；

  • 三要听别人的回答，并给予考虑，采取行动，重视别人的建议；

在学校里面，你正在上学时的老师会和你一起探讨一个问题，最后自己也会收获一些解决问题的能力。然而当工作时在很多情况下，身在领导位置的人面对下属的问题要么让下属自己解决，要么就直接会给出解决方案，而很少有领导者会想要引导你自己去想明白这个问题，因为这个世界总是喜欢方便快捷的人多。

但其实如果领导者能够充分发挥下属的能动性，将每一个雇员的所包含的有用的主意，观点，建议等等都考虑在内，这样每一个员工执行时有自己的理解，充满动力，最后的决策往往能够得到更好地执行，效率也会得到很好的提高。同时整个团体的凝聚力和能力也会有一些提升。

这让我想到了同宿舍的同学，她是的学习委员。对于的相关工作她是一个凡事都要自己再仔细过一遍，检查交上来的东西大体上有没有错误，比如交的实践作业，很word多人格式不对，但是她都自己把那些按照统一的格式改了一遍，明明是学生自己的作业，出现问题本就是应该自己改的，这样不仅能够提高自己这方面的能力，改进不足，也能够达到写实践报告真正的目的。而她这么一统一改，既掩盖了问题，让做作业的人不知道自己的错误，下次可能再犯；又平白给自己揽了很多活，看似尽到了学委的责任，帮助同学的学习，但其实在这一过程中完全没有考虑到做作业人真正需要的是什么。他们真正需要的是一次实践及实践报告带给他们的经验，而不是一次实践报告给他们加了多少分。这种带有个人英雄主义色彩的工作方式，我觉得是不适应团体需要的，也是不应该被提倡的。

《辛德勒的名单》观后感

发表于 2017-11-16 | 分类于 Movie

在战争中，人黑暗的一面会被放大，人善良的一面也会被放大。阿蒙在战争中变的残忍，凶狠，违背自己的内心，人种之分根深蒂固。辛德勒在一开始看到了战争带给他巨大的生意利益，不仅满足了自己以前做生意失败的借口（我觉得是借口，他生意真正成功很大一部分是史丹的原因），还让他拥有了很多财富，物质生活非常奢靡。但是渐渐他看到犹太人被大量杀害，而史丹间接救下留在工厂的犹太人对他真诚的感谢激发了他善良的一面，让人感受到在战争中，邪恶的不断膨胀中，善良也在不断膨胀。乃至他不惜花重金挽救上千个犹太人的生命，为生命，为人命而努力，坚持。

到了战争结束。由一开始把战争看做发财的机遇到最后不为战争制造一枚可用的子弹。他看到了战争的危害，侵蚀人性，夺走无辜人的生命。
他被史丹改变后，也想改变阿蒙。他在阿蒙喝醉不太清醒时说了一席话，试图使阿蒙能够“饶恕”别人，停止对无辜人的杀害。阿蒙第二天确实有了一些改观，然而却只持续了很短的时间，甚至半天都不到。我觉得在他醉酒时他内心世界的另一扇窗才会打开一点。阿蒙是个很复杂的人，就像在邪恶与善良的分界线上，从善良跨到了邪恶，但是不完全。

都说犹太人是世界上最聪明的民族，他们很会创造财富，这一点在史丹的身上体现了出来。在犹太人受到解放后，他们不知道往何处去，他们需要食物。将一片空地建造，发展成一个小镇，他们具有很强大的开创精神。

挽救了一个生命，就是挽救了世界。没错，当一千一百人的生命都被辛德勒挽救时，他们也救了辛德勒，同时建立起了一座小镇。辛德勒名单代表着至善，这份名单就是生命，名单的外围是一片可怕的深渊！辛德勒犹太人的狗带现在仍在生活着。

晋商之道

发表于 2017-11-09 | 分类于 Read

简述一下中国第一家票号：“日昇昌票号”的用人和风险防范。

用人四条基本法则：

经理负责制

俸股制

分为银股和顶身股。

其中顶身股的创制尤为机巧，保证了劳资关系和激励约束等方面的和谐制衡。银股分为正本和副本，即名义资本和实际资本。正本为股东的合约投资，以账期按股分红。

副本有二：一是东家、经理、伙计在结账期所分红利的 $提留$ ,票号专称“统事”或“获本”，不分红，只得息，不能随意抽取，以备防不测。二是东家在票号的存款。

这正是：“薪金百两是外人，身股一厘自己人”。对于现代的 $股份制改革$ 有一定的借鉴意义。

学徒制

初入票号的学徒须在总号经受为期三年的极为严格的学徒训练。

第一年干杂活，以考察品德为主，苦其心志，劳其筋骨，以除虚伪，节情欲，敦品行，贵忠诚，鄙利己，奉博爱，喜辛苦，戒奢华。

同人担保制

必须有同乡和殷实商户做担保，“用乡不用亲”。以及有类似于今天人事工作所提倡的“举荐问责”制度。当时的信用环境中商业经营很重要的一个手段就是担保（或联保）、相与（结盟），讲究的是“一荣俱荣，一损俱损”。人想学坏都难，有多少双眼睛八小时内外都盯着你呢。

严密的风险防范措施

异地汇兑两个关键风险点：一是汇票防伪，二是银两兑付。

汇票防伪

一张汇票涵盖了今天票钞管理的纸张、水印、编码、登记、验印、书写规范等多项技术手段。

密押技术：以不同汉语诗句中的不同单字来代指客户汇兑的金额和时间，（加之严格的内部管理制度）在后近百年内无一造假冒兑。

银两兑付

银色歌和平码歌，雷履泰的一大贡献就是确立了当时平色余利的恰当标准，成为票号一项相当好的收入。用现代银行业术语来表达，这就是典型的套利业务。

票号待解之谜

  • 密押如何在汇票上体现，如何传递，是否告知客户本人等关键问题；

  • 对于资金寸头问题，现在只知晓日升昌是利用 $浮动汇费、存贷利率差$ 及少量投资进行微调，至于如何调拨白银已解决各分店资金平衡难题也无从考证。

老母老婆落水谁需纠结

发表于 2017-11-04 | 分类于 Philosophy

呜噜学士相

“古时候的哲人曾经说过：人生在世，很重要的一点就是要智慧豁达，何谓智慧豁达呢？我以为就是没事儿别盼自己倒霉。

“当代大儒，我的老师、时任太傅的招财先生曾在他所著的《通州异闻志》中讲过一个故事。一日，招财先生的车队被一个读书人撞上。招老见其神情恍惚，惶惶然像要下流泪来，于是斥下持刀上前的护卫，询问原因。读书人忧虑地说：我近日总是担心遇到贼人对我妻子欲行不轨，如果我上前拦住贼人，自身就会被杀死，辜负了父母的养育之恩；如果我不上前，贼人就会得逞，折损妻子的名节，实在是我身为丈夫的失职。我日思夜想，但无论如何应对，总是面临死局，今天走在街上，脑子里想着这件事，以至于冲撞了老爷，请老爷点拨于我。

“招财先生听后，头发和胡子都竖起来，指着读书人痛斥道：糊涂至极。在歌舞升平之时，不去好好享受，却给自己出两难的题目，世上的事情还有比这更无谓的吗？古往今来，多少士人在 “母亲和女友落水时救哪个” 的问题前折戟沉沙，以至于家破人亡、妻离子散的也不为少见。身为读书人，不但不因未被问到这个问题而感到庆幸，反而给自己设下其他永远不能解决的问题，让脑海里的盗贼对付脑海里的自己，无论如何应对都面临僵局，就算是未曾上过学的蒙昧顽童也不会做出这样无谓的事情！读过圣贤书的人，却愚昧到这种程度，真是令人感到羞愧和吃惊。

“读书人听了招老的话，好像被一盆油酥从头上浇下来一样，脸上露出狂喜的神色，向招财先生恭恭敬敬地磕了三个头，嘴里发出“耶~耶~”的声音，手舞足蹈地跑远了。后来听闻此人找到当地精通军武的人为师，能舞动二百斤的石锁，七八个贼人也不能近身，从此不必担心遇到强盗的问题了。

“唉，招老所说的这件事情，并不是违背了古人 “居安思危” 的训导，而实在是因为读书人钻入了死胡同呀。世上的事情多种多样，具体的情况瞬息万变，试图想出一个具体的方法应对虚构出的局面，这难道不是犯了古人所说 “刻舟求剑” 的错误了吗？如果执着于“具体的方法”，而实际情况发生了变化，难道不是会像失去头领的羚羊一样手足无措，又好像被突然抢走狗咬胶的幼狗一样不知如何是好吗？与其苦苦思索具体的 “方法”，倒不如锻炼敏锐的头脑、健硕的身体、正直的心灵，这样在遇到危险的时候，能够凭着自己的本意出发，遵从自己的本心，又能够随机应变，难道不是最好的选择吗？但千千万万的人，把太多的精力耗费在 “方法 ”而非 “原则” 上，又怎么能立下功业，孝养双亲，荣耀祖先呢？人类的自大，从这些事情上，就可见一斑呀。”

摘自《呜噜学士书信笈.再拾》中《春日登香山渐入霾》一章。

在知乎上看到有人问一个问题：“面对女生被性侵，男生值不值得以性命为代价保护女生”，这是某个答者的回答，觉得很有智慧。其实若真的遇到类似情况，自己又无力让不法之徒在搏斗中输给你，我觉得还是要看应变能力和自己的本心了。但是，在没有遇到之前该做的事情就是力求活，而非死。所以，对于这种问题到底是哪种人才需要纠结呢？

消失的100日元

发表于 2017-10-31 | 分类于 Secret

今天看了一个数学谜题；“100日元在何处消失了？”

题目：三位顾客各出1000日元买了3000日元的东西。店主嘱咐店员说“东西稍稍旧了一点，可以便宜500日元卖”。店员考虑500日元三人无法均分，便将200日元放进口袋，交给顾客物品和300日元。

结果，顾客每个人拿出900日元，合计花了2700日元，由于店员放进口袋200日元，总共为2900日元。

那么100日元在何处消失了？

其实以前看到这种问题往往会想不通，也没有认真去想过，觉得就是比较绕而已（其实对于我的智商也确实有点绕😁）。但是我现在发现了一个方法，
就是用会计中的收付记账法；

1.顾客付出3000，店家收入3000；

2.店家付出2500，顾客收入2500(因为店员交给顾客价值2500的物品);

3.店家付出300，顾客收入300；

4.店家付出200，顾客收入200。

最后算的时候店家收入3000，付出3000；顾客也是收入3000，付出3000。根据“有收必有付，收付必相等”的原则可知，这笔交易没有问题。

原来这种题对于稍微学过会计知识的人来说是如此的简单。。。。
