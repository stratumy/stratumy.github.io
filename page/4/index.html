分清资产与负债——《富爸爸 穷爸爸》

发表于 2017-10-22 | 分类于 Read

富人买入资产；

穷人只有支出；

中产阶级买他们以为是资产的负债。

文字定义:

资产就是能把钱放进你口袋的东西；

负债是把钱从你口袋里取走的东西。

学校从来都是教如何花钱，却从来都没教过如何挣钱。为什么这样说呢，学校总是会传递给你一个好好学习，取得好成绩然后找一个高薪工作的信息，找到高薪工作则是为了有钱可花，可以买奢侈品，买大房子，开好车，这些都是花钱。人们通常认为找到工作，然后努力工作、发工资、买到东西，这些就是代表自己有资产了。

引用书上一个简单的例子，

  • 一个穷人或者尚未离开家的年轻人的现金流向图：

  • 下面这个是一个中产阶级的现金流向图：

  • 下面是一个富人的现金流向图：

这几张图虽然看起来只是简单反映了他们的金钱使用方向，但是却很鲜明地指出了穷人、中产阶级、富人三种人最后得到的资产与负债情况。

中产阶级在得到高薪水后往往想要住更大的房子，开更好的车，接着随之而来的就是生活上一些配套的奢侈品，最后以透支信用卡，向银行贷款等方式满足自己对房子，车子的要求。生活需求不断地增加，不断地买，不断地支出，不断地借钱，最终负债不断增加。他们把工资作为唯一的资产来源，最后买来的都是负债，但是他们把这些都当作是资产。因为我现在涉世未深，所以举得例子都是作者所举的，但是个人觉得很有道理。这些人往往会走进一个“老鼠赛跑”的陷阱中：早早背负了那些承受不起的债务，但是生活仍要继续，支出不会停止，自己的收入不会发生翻天覆的的变化，自己一直在循环着一个努力挣钱还钱的过程。他们往往是量入为出，这时候需要改变的就是财务观念和支出方式。正如富人都是先拥有资产再买奢侈品。

当你有额外的资产时你才有钱去进行投资，然而很多人从未有过产生真正收入的资产，他们的唯一收入来源就是他们的雇主，自己没有主动权。因此也就要把自己保护在安全范围之内。一些富人因为掌握了很好的财务知识加上能够抓住好的机遇会获得收入，有一些人支配资产的方式则没有这么有效率，比如一个中了彩票的人，在不久后不仅花光了所有的钱，还倒欠钱。因为他用钱买入的是负债而不是资产。

这个对资产与负债的定义与专业财务人员学到的不一样，但它能符合你的生活，并不是说专业财务知识不对或者是对生活没用，只是有时候人们可能忘记去用，或没有机会去学习了，从简单开始，以非专业但是必需的角度去传播。

本文所说的只是资产与负债在实际生活中的一种理解，或许还有更多的理解，从不同角度，不同的立场等等。

读《刻意练习》（一）前言

发表于 2017-10-06 | 分类于 Read

刚开始读《刻意练习》这本书，就先把前言总结一下吧，是从结构和内容两方面总结的：

结构

  • 这本书主要是讲刻意练习，前言中先从练习说起，对练习进行了解读，将练习分为天真练习和正确练习。

  • 而后又以练习引出了一万小时定律因为这个定律对天真还是正确的练习没有做明确，而仅仅以一万这个时间的量来指导人们进行联系，与本书的刻意形成了一定程度的对比，所以我觉得由此其实从另一个方面也引出了 刻意 这个话题。

对一万小时定律反驳，从语文的角度分析，我觉得他是从正面和反面两个方面来反驳的：

  • 不同专业领域所需时间不同。

  • 成功与练习时间不完全成正比。

  • 练习成果不与时间呈正相关。

  • 最后以“沙堆悖论”从反面驳斥了一万小时定律（可变边界）。

  • 让人稍微了解到刻意练习后，又提出了刻意练习的本质，从而引出一个概念：长时记忆。到这里就比较套路一点了，基本按照是什么，为什么，怎么做的顺序来写的。

  • 最后提到的隐性知识，我觉得是对刻意练习正确性一个抽象性的补充，其提出的情境学习我觉得就是对刻意练习的一个方法的补充，也就是对怎么做的一个补充。

内容

总的来说就是简要介绍本书的内容，目的，引起读者兴趣。我主要看到了以下几个概念：

  • 大脑的适应能力

  • 长时记忆以及方法·培养方向

  • 认知复杂性高低

  • 创造潜力

  • 情境学习

像花一样绚烂

发表于 2017-09-21 | 分类于 College

军训刚刚开始，真心是很累的，中午由于噪音睡不着，躺在床上歇了一会后就去取快递了。拖着疲惫的身体，打着雨伞，一步步试着走向百米微。因为昨天晚上淋着雨，跟着高德找它，但是最终还是没找到的我，所以那时确实只能是一步步地尝试性地走了。也不像晚上那么不敢走，想着反正也不知道怎么走就随便走吧。

清新靓丽本就应该是青春的颜色。取到快递后，离集合时间还早（想逛一逛校园多记点路），我就自己心怀诗意地撑着伞走在林荫大道上，看着这条道上稀少的行人和清新自然的美景莫名的一种惬意感涌上心头，身体也不那么疲惫了。散步，观景，沐风，观雨，赏心，悦目。两个学长骑车路过时爽朗地笑着，嘴里夸赞着这条路的好。他们从我身边经过时那份自然的欢快爽朗与朝气瞬间即感染了我。眼前的一切仿佛都变的更加明朗美好起来了。

填充色是那么的耀眼。经过击剑中心的时候，透过玻璃窗看到很多的奖杯矗立在架子上，早就听说我们学校的击剑队很牛X，今日一见果然名不虚传啊（说话能力有限，只能用这么老套的话来形容了O(∩_∩)O）。看到那么多奖杯的时候其实心里只想到了一句话：好厉害啊。然后就是一副看呆了的表情，想趁中午没人进去参观一下，看到擦地板的阿姨后，因为怕被赶出来我果断地走了（对，就是这么果断，哈哈 毕竟刚来的新生还是不要这么莽撞，主要还是有过这方面的教训）。但是奖杯真的是很耀眼，感觉有了这些奖杯后，那些击剑运动员应该很有成就感吧，希望我的青春也能像他们一样手握奖杯，外人面前光鲜，自己心里踏实高兴（偷笑）。

绿草茵茵，绿树盖顶，小花点缀其中，青春朝气作为营养，这方天地，尽管有限，却很值得体验一回，作为回忆收藏。

真是，一路上感觉草都生长得像花一样绚烂了。

不耐烦——试着习惯

发表于 2017-09-16 | 分类于 College

今天开学第一天，刚开始去学校因为东西多离公寓楼稍微有点远（其实学校相对来说很小），所以一开始就心情不太好，因为大汗淋漓的滋味真的很难受。然后接着一系列的事报到什么的都要顶着大太阳走一段路，还要排队，很是麻烦。最受不了麻烦的人就是我了吧，一路心情都比较低落，再加上学校没有预期的好，突然觉得有点后悔来到这里。

全程不耐烦的我，幸亏有姐姐在要不然我的生活能力的短板真是会被极其尴尬地暴露出来。

后来看到姐姐和妈妈都不嫌麻烦不嫌累地帮我收拾，我真的是觉得自己太不应该了，然后送走她们后就鼓起勇气和室友，老师，导生说话，试着帮老师收发登记。其实大学只是你人生的一次经历，每一次好的经历都是宝贵的。努力做到最好，不服自己，至少不让自己后悔，试着去习惯，去热爱，很多像你一样的人甚至还不如你的人都在不断地尝试，你还有什么理由不去呢……明天能够再勇敢一点吗？要是能够再谦和有礼，大方一点就更好了。

说说虚荣

发表于 2017-09-10 | 分类于 Read

我上高中的时候买过一本书《时文选萃》，不是经典名著，也不是畅销书，但是却觉得它很受用。这书是分辑出的，我就买了其中一本。上面有一篇文章是关于虚荣的叫做《聊聊虚荣》。

虚荣似乎是天生的，每个人都有。有虚荣之心不是什么坏事儿，但虚荣过了头就是坏事儿了。

有两个同学，一个高个，一个矮个，他们俩都是百米健将，不分伯仲。到了开运动会的前一天晚上，高个的怕输给矮个的，就趁矮个熟睡时，将他的头发剪成了“菜花秃子”，想以此让他不好意思参加比赛。但第二天，矮个的就顶着那个“菜花秃子”头赢得了比赛。

孟德斯鸠说，有些人的虚荣心，竭力使别人不愉快，想借此引起别人的钦佩，他们设法要出人头地，结果反而更加不如人。结合现实想想也真是，多少人一时将别人“剪”成了“菜花秃子”，最终可能是输得自己练“秃子头”都保不住了。

当你在别人面前嘲笑显摆的时候就暴露了你的愚蠢，无论你看过多少书，有多么高的地位，多么好的声望。

实际上，虚荣有无都不会有什么额外的好处或坏处，但很多人都解不了这个套，知识分子不能，名人大家也不能。波普说，每一个人的虚荣都是和他的愚蠢程度相等的。放弃虚荣吧，免得让自己成为一个愚蠢的人。

少年奇幻漂流记的最大感受

发表于 2017-09-09 | 分类于 Movie


发光的海


惆怅的 孟加拉虎

  • 派一直相信上帝，但他最后还是感谢他父亲给他上的那一课，使他终身受益，并救了他的命。

  • 他最后还是感谢理查德的存在让他时刻清醒，让他相信可以活下去，最后他能够生存下去与这只孟加拉虎的存在密切相关，可以说是必要条件。

  • 多掌握一门技能总是好事，蔡康永有一段话很多人应该听过（这里就不多说了）和这个想法差不多。不管是15岁，18岁还是28岁都这一理论都适用。

  • 人不一定胜天，但可以胜过能够胜过的东西，这就是尽全力。
