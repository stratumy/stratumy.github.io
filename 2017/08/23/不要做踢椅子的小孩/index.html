不要做踢椅子的小孩

发表于 2017-08-23 | 分类于 Read

  • 蹒跚学步的小孩踩着椅子摔下来后有可能会恶狠狠地踹椅子一脚，怪怨椅子让她摔倒，似乎这样就能减轻他的疼痛。小孩子这样做无可厚非，但如果成人之后还这样做的话就是一种不负责任的表现，也是不正常的。

    • 自古以来，人们就普遍存在一种诿过于人的不良倾向。偷吃了禁果的亚当，却让可怜的夏娃一人承担罪责：“要不是那妇人引诱我，我怎么会吃呢？”把责任推给别人是比自己承担起来是容易得多的。一个人成熟的第一步就是负起应负的责任，如果作为一个成年人再去踢椅子的话绝对会让人笑掉大牙。

  • 对于那些不成熟的人来说，理由是无处不在的，甚至会从先祖·政府乃至全社会找一个能够让自己逃避的理由，最后甚至责怪自己压根就不应该来到这个世界。就像我以前把自己成绩不好归结到家族上来，由一开始有这个想法的惊讶到后来欣然接受并把它当作一个合理的理由去回应别人的自如。这就是一个人不成熟的表现。

    • 再举一个例子。我们会在电视上常常看到富二代常常抱怨父母不关心他们的生活，只是一味地拿钱给他们，父母亲的成功妨碍了他们的发展。最后我们就会理所应当地认为是他们的父母造成了他们性格上的不完美，行为上的不正常等等。奇怪的是，乔治·华盛顿他也同样有着良好的出身，富裕的家庭环境，可是他却成为了美国第一任总统，他从未抱怨父母给他造成了什么心理压力；亚伯拉罕·林肯，他幼年的物质田间极为匮乏，一切都只能靠自己双手获取，这也并没有成为他前进路上的绊脚石，而且也从没想过要去责怪任何人。1864年，他曾说过这样一句话：“我对美国人民 基督教世界 历史，还有上帝最后的审判，均负有责任。”这句话可以说是人类史上最勇敢的宣言了。

  • 对于心理治疗我认为还是存在科学上的解释的，因此在这里不对其进行反驳。

  • 在很早以前，不论中西都流行着一种逃避现实的说法：星象学。但是，莎士比亚在《凯撒大帝》一剧当中却借凯撒之口写下了如下的话：“亲爱的布鲁塔斯，这过错并非由于我们所属的星辰，而是我们向命运妥协了。”借口就这样被揭穿了。
